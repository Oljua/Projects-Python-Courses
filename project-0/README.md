## <a name="Задача_проекта)"></a> Задача проекта
Необходимо написать программу, угадывающую число от 1 до 100.

## <a name="Описание_проекта)"></a> Описание проекта
На вход в програму задается 1000 случайных чисел. Необходимо написать алгоритм угадывающий число с минимальным количеством попыток.
На выходе необходимо вывести среднее число попыток за которое алгоритм угадывает "загаданное" число.
