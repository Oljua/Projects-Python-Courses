# Проекты курса Data Science
---
#### Данный репозиторий представляет собой реализованые проекты во время прохождения специализации Data Science.
---
В рамках курса проходит обучение:  
— Python и ее библиотек для анализа данных;  
— основам методов машинного обучения (обучение с учителем, без учителя), решение и оптимизация алгоритмов распространенных типов задач (регрессия, классификация, кластеризация);  
— Deep Learning и обучение нейронных сетей (вкл. сверточные, рекуррентные) для решения и оптимизации всех основных задач, включая задачи компьютерного зрения и работу с естественным языком;  
— Data engineering (вкл. сбор, подготовку данных);  
— применение навыков в production, выбирая оптимальные решения для целей бизнеса, управление Data Science проектом.
